CI / CD para proyecto en Net Framework 4.8

2023

El proyecto consiste en la implementación de CI/CD para una solución desarrollada en Net Framework.

Para poder llevar a cabo la tarea se desplegó y configuro desde cero Jenkins, Sonarqube y se armaron scripts Powershell customizados

El SCM elegido fue Github pero, al ser un software desarrollado en Net Framework no fue posible implementar Github Actions. Se opto por hacerlo en Jenkins corriendo en Windows Server 2019.

Se utilizan los Pipelines de Jenkins en un Job declarativo trayendo el codigo desde Github y almacenándolo en un repositorio local en la VM de Jenkins.

Se envían mensajes a través de Telegram y Google Chat para informar el estado del despliegue.

El resultado de la compilación es enviado para su análisis a Sonarqube.

Los artefactos son almacenados en el repo local con información de hash de archivo (SHA256).

Este proyecto aportó un enfoque velocidad en el despliegue, automatizo los tests y agilizo la comunicacion entre QA y Desarrollo.

Diseño y Despliegue de Aplicaciones con Docker

2021

Este proyecto me llevo a diseñar todos los entornos para desplegar aplicaciones que luego serian Core en el dia a dia de la organización.

Se generaron 2 entornos DESA y PROD, el primero OnPrem y el segundo en la nube.

El SCM elegido fue Github. No se implementaron Actions.

Para analizar los contenedores se utilizo Snyk.

Se utilizo OpenLDAP para proveer Identidad a las aplicaciones.

Toda la infrestructura esta siendo monitorizada via Zabbix.

Se implementa un esquema simple de Backups mediante BASH + CRON + RSYNC.

Se creo una integración en python que genera y actualiza tickets en ITop a partir de Problems de Zabbix

La organizacion no contaba con Herramientas de Tickets, Gestion, Mensajeria, ni sitio oficial. Esta implementacion mejoró la productividad y el control en las áreas de Operaciones y Gestión de Proyectos. RocketChat aportó un medio de comunicación instantánea organziacional ya que antes se manejaban via WhatsApp
